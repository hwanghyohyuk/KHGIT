팀명 : 
프로젝트명 : 식당 선주문 예약 시스템
기획 의도 : 
	1. 바쁜 현대인들의 점심시간을 효율적으로 활용하기 위해서
	2. 식당 웨이팅을 최소화를 위해
	3. 코딩능력 향상 

프로젝트 기능 요구사항
1. 로그인
2. 회원가입
3. 회원정보 수정
4. 회원 탈퇴(지점관리자 포함)

4. 회원을 위한 기능
-가게 검색
-업종별 리스트화
-즐겨찾기(해당 가게의 메뉴 등록)
-메뉴검색(해당 메뉴에 대한 가게정보 리스트화)
-주문리스트 작성
-주문
-주문내역확인(해당가게에서 직접확인할 내용)

5. 지점관리자를 위한 기능
-주문알림
-주문 거절 (재료소진, 만석, 마감시간)
-지점 정보 수정
-가게 만석일 시 가게 OFF 
-메뉴 OFF
-메뉴 등록 수정 삭제
-주문 내역 통계

6. 서버
-로그인
-회원 관리
-가게 리스트
-메뉴 리스트
-주문
-로그 저장
