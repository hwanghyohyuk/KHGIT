2. 자바의 특징을 3가지 이상 기술하시오

	1. 객체지향 언어
	2. OS에 독립적
	3. 자동 메모리 관리 (Garbage Collection)
	4. 소스 오픈, 가볍다, 멀티 스레드 지원
	
3. Java의 Namming Rule이란?

	1. 예약어를 사용할 수 없다.
	2. 숫자로 시작해서는 안된다.
	3.특수문자는 '_' 와 '$' 만 허용된다.
	
5. import
	
	서로 다른 패키지에 있는 클래스를 참조하기 위함이다.
	소스 파일에 여러 번 선언이 가능하다.
	클래스 선언 이전에 명시한다.
	패키지 선언 다음에 명시한다.
